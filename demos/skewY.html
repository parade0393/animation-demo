基础概念

坐标系统

Transform 变换

translate() 平移 rotate() 旋转 scale() 缩放 skew() 倾斜 skewX() X轴倾斜 skewY() Y轴倾斜

skewY() - Y轴倾斜

沿 Y 轴倾斜元素，创建垂直倾斜效果

语法说明

transform: skewY(angle);
  • angle - Y 轴倾斜角度
  • 正值：元素向下倾斜
  • 负值：元素向上倾斜
  • 等价于 skew(0, angle)

示例 1：向下倾斜

skewY(20deg) 将元素向下倾斜 20 度

20°
.box { transform: skewY(20deg); /* 向下倾斜 20 度 */ }

示例 2：向上倾斜

使用负值向上倾斜

-20°
.box { transform: skewY(-20deg); /* 向上倾斜 20 度 */ }

示例 3：大角度倾斜

skewY(30deg) 创建更明显的垂直倾斜

30°
.box { transform: skewY(30deg); }

提示：skewY 通常与 skewX 组合使用，创建更复杂的倾斜效果
